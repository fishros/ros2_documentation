Rolling Ridley 是 ROS 2 的滚动开发版本。

警告

Rolling Ridley 持续更新，并且会不时进行就地更新，其中可能包含破坏性变更。它用于 ROS 2 的开发，以及维护者希望他们的软件包发布并准备好下一个稳定版本的情况。我们建议大多数 ROS 2 用户使用最新的 稳定版本。

有关详细信息，请参阅 REP-2002

目前支持的平台

Rolling Ridley 当前支持以下平台：

一级平台：

  • Ubuntu 22.04 (Jammy): amd64 和 arm64

  • Windows 10（Visual Studio 2019）

Tier 3 平台：

  • Debian Buster (10)：amd64，arm64``和``arm32

  • Mac macOS 10.14 (Mojave)

  • OpenEmbedded Thud（2.6）/ webOS OSE：arm32 和 x86

本版本的新功能和变更
