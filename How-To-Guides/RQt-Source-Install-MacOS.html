在 macOS 上从源代码构建 RQt [763]

本页提供了在 macOS 上从源代码构建 RQt 的具体信息。在继续查看 RQt 源码安装 页面之前，请按照这些说明进行操作。 [764]

系统要求 [740]

RQt 在 macOS 10.12 上受支持，但似乎在 10.13 上也可以工作。 [765]

依赖项 [766]

RQt 包的主要依赖项是 sip 和 PyQt5。将来可能支持 PySide2。 [767]

安装依赖项 [199]

$ brew install sip pyqt5
$ brew install graphviz
$ python3 -m pip install pygraphviz pydot
$ brew link --force qt

这是最快的解决方案，但在升级 Qt 或其他包需要 Qt 4 的情况下可能会出现问题。另一种选择是更新您的 PATH 和 CMAKE_PREFIX_PATH，将 Qt 安装位置包括在内： [768]

$ export PATH="$(brew --prefix qt)/bin:$PATH"
$ export CMAKE_PREFIX_PATH="$(brew --prefix qt):$CMAKE_PREFIX_PATH"
