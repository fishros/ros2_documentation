GurumNetworks GurumDDS

rmw_gurumdds是使用GurumNetworks GurumDDS实现的ROS中间件接口。有关GurumDDS的更多信息，请访问我们的网站：https://gurum.cc/index_eng

先决条件

以下说明假设您已完成 通过Debian软件包安装ROS 2 或 在Ubuntu Linux上构建ROS 2 中的 '环境设置' 过程。

rmw_gurumdds 需要 GurumDDS-2.8.x 的版本。GurumDDS 的 Debian 包可以在 Ubuntu 上的 ROS 2 apt 仓库中获取。GurumDDS 的 Windows 二进制安装程序即将支持。

GurumDDS 需要许可证。请参阅下一页：https://gurum.cc/free_trial_eng.html

在申请试用许可证后，请从许可证主页下载许可证。获取许可证后，请将其移至以下位置。

DDS 版本

许可证位置

<= 2.7.2860

/etc/flame

>= 2.7.2861

/etc/gurumnet

2.8.x

/etc/gurumnet

安装软件包

最简单的方法是从ROS 2的apt存储库安装。安装ros-humble-rmw-gurumdds-cpp时，也会安装gurumdds-2.8。

sudo apt install ros-humble-rmw-gurumdds-cpp

从源代码构建

从源代码构建也是安装的另一种方式。

首先，在ROS 2工作空间源目录中克隆rmw_gurumdds。

cd ros2_ws/src
git clone https://github.com/ros2/rmw_gurumdds -b humble ros2/rmw_gurumdds

然后，安装GurumDDS所需的必要软件包。

cd ..
rosdep install --from src -i --rosdistro humble

最后，运行colcon build。

colcon build --symlink-install

切换到rmw_gurumdds。

通过指定环境变量，从其他rmw切换到rmw_gurumdds。

export RMW_IMPLEMENTATION=rmw_gurumdds_cpp

参见：使用多个RMW实现

运行发布者和订阅者

现在运行 talker 和 listener 来测试GurumDDS。不要忘记使用设置脚本来设置环境。

ros2 run demo_nodes_cpp talker

ros2 run demo_nodes_cpp listener
