RTI Connext DDS

RTI Connext DDS 6.0.1的仅库版本可以根据Debian/Ubuntu Linux（amd64）平台的 安装指南 进行安装，仅限于`非商业许可证 <https://www.rti.com/ncl>`__。

RTI Connext DDS的完整套件安装可适用于许多其他平台，供大学、评估或购买使用。此安装包括诊断工具、分层服务和安全性。有关安装详细信息，请参见下文。

RTI大学计划

大学研究人员和教室用户可能有资格通过 RTI大学计划 获得免费的学术许可证。这包括一年（可续订）的Connext DDS Secure完整版本许可证，其中包含诊断工具和分层服务。可以在 这里 找到大学许可证申请。

RTI Connext DDS评估

要安装RTI Connext DDS 版本6.0.1 评估版：

  • 访问 RTI 免费试用（6.0.1）网站。

  • 下载与您的环境匹配的版本。

  • 请联系 license@rti.com 以获取评估许可证。

  • 通过运行安装程序安装 RTI Connext 6.0.1。安装完成后，它将运行 RTI Launcher。

  • 如有需要，使用 RTI Launcher 安装许可证文件（rti_license.dat）。该启动器也可用于启动诊断工具和服务。
