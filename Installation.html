你应该选择哪种安装方式？ [7543]

从二进制包或源代码安装都可以获得完全功能和可用的ROS 2安装。选项之间的差异取决于您计划如何使用ROS 2。 [7544]

**二进制包**适用于一般用途，并提供了一个已构建的ROS 2安装。这对于希望立即开始使用ROS 2的人来说非常方便。 [7545]

Linux 用户有两个选项来安装二进制软件包： [7546]

  • Debian 软件包 [7547]

  • "fat" 存档 [7548]

从 Debian 软件包安装是推荐的方法。它更方便，因为它会自动安装所需的依赖项。它还会随着常规系统更新而更新。 [7549]

然而，你需要 root 访问权限才能安装 Debian 软件包。如果你没有 root 访问权限，"fat" 存档是下一个最佳选择。 [7550]

选择从二进制软件包安装的 macOS 和 Windows 用户只有 "fat" 存档选项（Debian 软件包只适用于 Ubuntu/Debian）。 [7551]

**从源代码构建**适用于希望更改或明确省略ROS 2基础部分的开发人员。它也推荐用于不支持二进制文件的平台。从源代码构建还提供了安装ROS 2绝对最新版本的选项。 [7552]
