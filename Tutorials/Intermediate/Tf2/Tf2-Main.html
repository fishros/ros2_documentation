tf2 [待校准@14657]

Many of the tf2 tutorials are available for both C++ and Python. The tutorials are streamlined to complete either the C++ track or the Python track. If you want to learn both C++ and Python, you should go through the tutorials once for C++ and once for Python. [待校准@14658]

Learning tf2 [待校准@14661]

  1. Introduction to tf2. [待校准@14662]

    This tutorial will give you a good idea of what tf2 can do for you. It shows off some of the tf2 power in a multi-robot example using turtlesim. This also introduces using tf2_echo, view_frames, and rviz. [待校准@14663]

  2. Writing a static broadcaster (Python) (C++). [待校准@14664]

    This tutorial teaches you how to broadcast static coordinate frames to tf2. [待校准@14665]

  3. Writing a broadcaster (Python) (C++). [待校准@14666]

    This tutorial teaches you how to broadcast the state of a robot to tf2. [待校准@14667]

  4. Writing a listener (Python) (C++). [待校准@14668]

    This tutorial teaches you how to use tf2 to get access to frame transformations. [待校准@14669]

  5. Adding a frame (Python) (C++). [待校准@14670]

    This tutorial teaches you how to add an extra fixed frame to tf2. [待校准@14671]

  6. Using time (Python) (C++). [待校准@14672]

    This tutorial teaches you to use the timeout in lookup_transform function to wait for a transform to be available on the tf2 tree. [待校准@14673]

  7. Traveling in time (Python) (C++). [待校准@14674]

    This tutorial teaches you about advanced time travel features of tf2. [待校准@14675]

  2. Debugging tf2 problems. [待校准@14679]
