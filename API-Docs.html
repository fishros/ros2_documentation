所有ROS 2软件包的文档都与其相关信息一起托管在`ROS Index <https://index.ros.org/>`_上。搜索软件包将提供它们的信息，例如发布的发行版、``README.md``文件、URL和其他重要元数据。可以在此处找到索引上每个发行版的所有软件包列表：这里。在感兴趣的软件包的右侧，可以使用"API Docs"按钮找到自动生成的API文档。

您可以在Humble发行版中使用以下链接找到客户端库的API文档：

  • rclcpp - C++客户端库

  • rclcpp_lifecycle - C++生命周期库

  • rclcpp_components - C++组件库

  • rclcpp_action - C++动作库

可以在此处找到Humble软件包文档的原始列表 may be found here.
